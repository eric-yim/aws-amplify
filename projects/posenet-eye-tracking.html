Controlling Devices with Your Eyes

Imagine controlling devices with just our eyes. This could have many uses from mobility challenges to augmented reality enhancements. We can now do so with a bit of Machine Learning.

Here's the broad overview: we use PoseNet to detect eyeballs; then we use a custom model to classify eyeball direction.

PoseNet for Detecting Body Parts

PoseNet, developed for Tensorflow JS, is a mdel great at detecting human body parts on screen. The output of the model contains body part labels: leftHip, rightWrist, leftKnee, etc. Here we just take the leftEye and rightEye locations and crop out a square to be used for the next model.

Eyeball Classification

We've trained a custom convolutional neural network to classify 5 eye directions: straight, up, down, left and right.

We throw in some engineering tricks for training a robust model such as random cropping, horizontal flipping, and random color alterations. Then we convert the Tensorflow model to a Tensorflow JS model. The resulting file is a json file that just needs to be hosted on the internet, and it can be loaded using javascript.

The resulting models are called in javascript and are then loaded in the browser. The models use the camera and hardware available. For example, when visiting the site with a GPU enabled desktop, we can see an FPS of 30, whereas when visiting the site on a mobile phone, we can see an FPS of 1-3.
